# Ironhack Early Project - Circle Agency
*Primer proyecto del bootcamp Desarrollo Web Full Stack + IA de Ironhack - Circle Agency website*

## Elementos del sitio web
* Landing page con una breve presentación de la empresa, vistas previas de algunos proyectos realizados, servicios ofertados y formulario para unirse al servicio de newsletter
* Página de presentación de proyectos, con un espacio prominente para mostrar el proyecto seleccionado y un espacio más reducido de tres elementos combinados para las vistas previas del resto de proyectos
* Página de contacto con formulario para enviar mensajes


## Tareas realizadas
* Creación de la estructura HTML de las tres páginas
* Creación de una hoja de estilo .css para las tres páginas
    * Diseño responsive para adaptar la visualización a dispositivos de hasta 800 píxeles de resolución
* Creación de la lógica de manipulación del DOM en JavaScript
    * Uso de funciones asíncronas para recuperar información de un servidor mediante API
    * Manipulación del DOM para implementar la información recuperada
    * Recopilación de los datos del formulario
        * Confirmación de la idoneidad de los datos introducidos
        * Control de introducción de espacios innecesarios
        * Visualización de los datos en consola
        
*  Despliegue en [Netlify](https://dsmarrero-ironhack-early-project.netlify.app/).
